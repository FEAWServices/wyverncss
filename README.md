# WyvernCSS

AI-powered CSS styling for WordPress Gutenberg blocks.

## Description

WyvernCSS helps you style WordPress blocks using natural language descriptions.

## Installation

1. Download the latest release ZIP file
2. In WordPress, go to Plugins → Add New → Upload Plugin
3. Select the ZIP file and click Install Now
4. Activate the plugin
